Bingo 就业管理系统

摘要：

随着大学毕业人数和企业数量的增加，就业关系变得极其复杂。如果采用手工管理这些信息，不但耗时而且不便于分析。通过开发Bingo就业管理系统，学校可以准确了解就业情况，对此分析，作出决策；学生可以及时了解招聘信息，投递简历，完成就业；企业可以随时发布招聘信息，查看简历，招聘学生。因此Bingo就业管理系统可以有效解决就业信息管理的问题，方便学校、学生以及企业。

关键词：

就业；管理；招聘；学校

项目演示：

www.bingo.wisyoong.com

1 引言

随着大学毕业人数和企业数量的增加，就业关系变得极其复杂。学校渴望了解详细和完整的信息，保障学生的就业，甚至基于就业信息对今后发展作出战略性决策；学生渴望获取及时的招聘信息，投递简历，完成就业；企业渴望获取可靠的学生信息，聘请优秀人才。这些需求就对就业信息管理提出了新的要求。
采用手工管理就业信息，学校需要手动收集学生的就业信息、学生需要手动打印自己的简历信息、企业需要手动收集学生的简历，这些不但耗时而且不便于分析。
显然采用手工管理就业信息遇到瓶颈，已不能满足学校、学生、企业的需求。而当今日益发展的计算机技术恰好能满足这一需求，所以，就业管理系统就成了必需。该就业管理系统应该为学校、学生、企业提供充分的信息管理和方便的查询途径，可以方便、快捷、准确地获取学生信息、简历信息、企业信息、招聘信息和就业情况，使学校、学生、企业更加了解当前的就业情况。

2 需求概述

2.1 功能需求

一、 管理员（学校）用户

  1. 用户管理：查看、审核和删除用户。
  2. 学生信息：查看、修改和删除学生的信息。
  3. 企业信息：查看、修改和修改企业的信息
  4. 招聘信息：审阅招聘信息，删除不合规的招聘信息。
  5. 就业信息：查看学生的就业情况。
  6. 留言管理：查看学生、企业的留言，并对他们的留言进行回复。

二、 学生用户

  1. 简历管理：根据个人信息创建多个简历，删除简历，修改简历。
  2. 招聘信息：查看企业发布的招聘信息，从已创建的多条简历中选择一个并。
  3. 就业信息：查看已投递简历的招聘岗位、招聘公司、应聘状态以及取消应聘。
  4. 留言：给管理员留言并查看管理员回复。

三、 企业用户

  1. 招聘信息：发布、修改和删除招聘信息
  2. 就业信息：查看招聘信息的投递情况、查看简历、以及招聘学生。
  3. 留言：给管理员留言、查看管理员回复和其他用户留言。

2.2 界面需求

随着人们生活水平的提高，人们的审美日益提高；随着科技的发展,显示器的分辨率不断提高，显示器尺寸也各不相同；随着用户数量的增加，软件的数据量不断增加。这些都对软件的界面提出了新的要求。所以，软件的界面要美观，符合人们的审美要求；要兼容，适配不同的显示器，保证在大部分情况下，软件界面在不同窗口尺寸、不同屏幕尺寸、不同屏幕分辨率下,能正常显示；要简洁，放入庞大的数据，使用户能尽快找到需要的数据。一个好的界面能提高软件执行效率，能节省用户使用软件的操作时间，能使人们的心情舒畅，提高工作效率。

3 界面设计

受谷歌Material Design的启发，该软件界面的边栏采用抽屉式设计，需要的时候显示，不需要的时候可以把它收起来，节省屏幕空间，如图3-1和图3-2所示。受Android的Toast和Mac OS的启发，该该项目加入了消息提醒框插件，当完成相关操作或出现错误时，在窗口的右下角会弹出消息框，如图4-1右下角，这样不仅简洁美观，而且减轻了服务器的负担。经过查阅大量相关资料，本系统的HTML标签的名字属性采用统一的命名方式，统一采用小写，统一采用“-”（减号）作为单词的分隔符号，比如<input name=”email”/>。经过精心调试，保证在大部分情况下，软件界面设计在不同窗口尺寸、不同屏幕尺寸、不同屏幕分辨下,能正常显示。

  4. 点击侧边栏的“招聘信息”进入招聘信息页面；
  5. 点击工具栏中的加号进入添加招聘信息页面；

8 结束语

经过以上详细的设计与架构，终于完成了毕业选题系统的开发。我主张软件产品不仅功能要完善，而且界面要好看，因为这些高要求，这个作品花了我大量时间。受Apple设计的启发，该软件采用先设计后工程的顺序完成，也就是先设计软件界面，再实现软件功能的模式。
该软件不仅界面和功能花了我大量功夫，而且在代码规范方面也花了不少心思。经过查阅大量相关资料，采用了比较规范的数据库命名方式，比如本系统的HTML标签的name属性采用统一的命名方式，统一采用小写，统一采用“-”（减号）作为单词的分隔符号，比如<input name=”email”/>。再比如数据库里的内容，命名一律采用小写英文，多个词的名称用“_”(下划线)分隔，外键用数据表名加列名的形式命名等等。登录采用邮箱和手机号登录和注册，方便用户记住账号。
这次毕业设计的制作，提高了我对知识的整合能力，提高了我们的实战能力，是一个很好的锻炼机会。也让我明白了自己所学的知识在真实的生产环境中，还远远不够，还需要不断学习，提高自己的能力。
